Sobre o projeto


O presente projeto foi feito com o intuito de praticar e demonstrar minhas habilidades com HTML, CSS e JavaScript. Esse projeto trata-se de um site para um PetShop. Através dele, é possível visualizar e encomendar os produtos vendidos.


Para encomendar um produto, basta abrir ele e clicar no botão "Pedir por WhatsApp". Feito isso, o usuário é direcionado para uma conversa com o vendedor no WhatsApp, automaticamente os dados do produto selecionado são transferidos para o chat do app. Após isso, basta enviar a mensagem e negociar com o vendedor.


Para ver a localização da loja física do pet shop, basta clicar em "Loja física" que está localizado no topo do site. Como a empresa é fictícia, escolhi uma ilha praticamente vazia.


A página inicial do site apresenta alguns cards com animais, os quais podem ser selecionados para visualizar produtos direcionados ao tipo de animal selecionado (como rações, petiscos, brinquedos e gaiolas).
